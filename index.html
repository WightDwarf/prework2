Prework Study Guide

Open the Console to See What's Happening

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The header and footer elements work like in a word processing program, and represent the top and bottom of the webpage.
  • Titles for paragraphs can be represented by h1-h6, in corresponding height.

CSS

  • A margin indicates how much space we want around the outside of the element.
  • A padding indicates how much space we want around the content inside of the element.
  • Colors can be represented by color words or by hexadecimal values.
  • Defining classes helps format only defined sections.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pulls the current version of the main branch to your branch
  • git add -A: adds all changed files to Git
  • git commit: moves files from gitbash to the repository
  • git push origin: creates a pull request for your new code in the repository

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An array can store multiple variables in it.
  • For loops can make the same code perform itself to the defined number of times.
  • console.log will print the appropriate result in Chrome dev tools Console tab.
